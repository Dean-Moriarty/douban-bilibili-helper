# 🎬 豆瓣 B 站解说助手

## 📌 介绍
**豆瓣 B 站解说助手** 是一款 **Chrome & Edge 浏览器插件**，它可以在 **豆瓣电影/书籍页面** 上 **自动推荐 B 站的解说视频**，让你无需手动搜索，一键查看相关解说内容！

🌟 **支持 “换一批” 功能**，每次点击都会刷新新的解说视频！  
🚀 **适用于 Chrome 和 Edge**，让你的观影体验更丰富！  



---

## 🎯 功能特点
✅ **自动匹配**：获取豆瓣电影/书籍名称，在 B 站搜索相关解说视频  

✅ **嵌入式推荐**：在豆瓣页面直接显示 B 站解说，**风格与豆瓣保持一致**  

✅ **封面 + 信息**：每个视频包含 **封面、标题、作者、播放量、弹幕数**  

✅ **一键跳转**：点击 **封面或标题**，即可跳转到 B 站观看完整视频  

✅ **“换一批”功能**：想看更多解说？**点击换一批**，获取新的解说视频  

✅ **“去 B 站查看更多”**：快速跳转 B 站查看更多相关解说

## 🛠️ 安装方式（插件商店还在提交审核中，暂不可用）

### **💡 方法 1：从 Chrome 商店安装**
👉 **[Chrome 商店插件链接](https://chrome.google.com/webstore/detail/豆瓣-b站解说助手/你的插件ID)**（审核通过后更新）

### **💡 方法 2：从 Edge 商店安装**
👉 **[Edge 插件商店链接](https://microsoftedge.microsoft.com/addons/detail/你的插件ID)**（审核通过后更新）

### **💡 方法 3：手动安装**
1. **下载本项目**（点击右上角 `Code` -> `Download ZIP`）  
2. **解压 ZIP 文件**  
3. **打开 Chrome 浏览器**，在地址栏输入：
chrome://extensions/
4. **开启“开发者模式”**
5. **点击“加载已解压的扩展程序”**
6. **选择解压后的插件文件夹**
7. **安装完成！**

## 🎬 使用方法
1. **打开豆瓣电影/书籍页面**，如：
- 电影：[https://movie.douban.com/subject/1292052/](https://movie.douban.com/subject/1292052/)
- 书籍：[https://book.douban.com/subject/4913064/](https://book.douban.com/subject/4913064/)  
2. **查看 B 站解说推荐**（显示在“在哪儿看这部电影”模块下方）  
3. **点击解说视频**，跳转 B 站观看  
4. **点击“换一批”按钮**，刷新获取新的解说视频  
5. **点击“去 B 站查看更多”**，跳转 B 站搜索更多解说视频

## 📌 常见问题
**Q: 为什么 B 站解说推荐没有显示？**

🔹 请确保 插件已启用，然后 刷新豆瓣页面。如果仍然无法显示，可能是该电影/书籍没有相关解说。

**Q: 为什么点击“换一批”后，还是显示相同的视频？**

🔹 可能是 B 站的搜索结果较少，请尝试 点击“去 B 站查看更多”，在 B 站上查看更多解说。

**Q: 如何卸载插件？**

🔹 打开 Chrome 扩展管理页面（chrome://extensions/），找到 豆瓣 B 站解说助手，点击 “移除” 即可。
